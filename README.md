# CRUD de Gerenciamento de Departamentos

Este repositório apresenta um exemplo prático das operações CRUD utilizando JavaScript puro e a API de Armazenamento na Web (Web Storage).

## Objetivo do Projeto

O propósito deste projeto é servir como um guia para estudantes interessados em aprender sobre:

- Implementação das operações básicas de CRUD (Create, Read, Update, Delete).
- Utilização da API de Armazenamento na Web para persistência de dados localmente no navegador.

## Branches Disponíveis

Para facilitar o acompanhamento do progresso do projeto, as seguintes branches estão disponíveis:

1. `inicio`: Projeto inicial vazio e configurado.
2. `cadastrar`: Funcionalidade para cadastrar novos departamentos (CREATE).
3. `listar`: Funcionalidade para listar todos os departamentos cadastrados em uma tabela HTML.
4. `visualizar`: Funcionalidade para visualizar detalhes de um departamento específico na tabela (READ).
5. `editar`: Funcionalidade para atualizar dados de um departamento existente na tabela (UPDATE).
6. `excluir`: Funcionalidade para excluir um departamento da lista (DELETE).

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
